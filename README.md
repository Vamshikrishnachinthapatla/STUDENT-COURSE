# Student & Course Management System

A Django web application with REST APIs that supports authentication, role-based access, CSRF-protected web forms, and token-based API authentication.

## Features
- User registration and login
- Role-based access (Admin & Student)
- Course management (Admin)
- Student enrollment
- REST APIs using Django REST Framework
- Token authentication
- Pagination and search
- CSRF protection for web forms

## Tech Stack
- Python
- Django
- Django REST Framework
- SQLite
- HTML, Bootstrap

## API Endpoints
- POST /accounts/api/register/
- GET /api/courses/
- POST /api/enroll/
- GET /api/my-courses/

## How to Run
1. Clone the repo
2. Create virtual environment
3. Install requirements
4. Run migrations
5. Start server
